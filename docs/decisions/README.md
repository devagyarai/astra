# Architecture Decision Records (ADRs)

This folder contains Architecture Decision Records (ADRs). ADRs document the context, consequences, and ultimate choices made regarding significant architectural transitions.

## Naming Convention

ADRs should follow a sequential numbering scheme accompanied by a descriptive, kebab-cased title.

### Examples:
- `ADR-001-Local-First-Architecture.md`
- `ADR-002-NextJS-App-Router.md`
- `ADR-003-Astra-Design-System.md`
