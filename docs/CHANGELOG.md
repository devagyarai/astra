# Changelog

All notable changes to Astra will be documented here.

## [Unreleased]
### Added
- **Decision Intelligence Engine**: Real-time deterministic analysis of your decision framework.
- **Smart Insights**: Right sidebar now dynamically displays Confidence, Completion, Risks, Strengths, Weaknesses, and Next Best Action.
- **Dashboard**: A comprehensive overview of all your decisions, including metrics like Average Confidence.
- **Timeline**: A chronological list of all decisions stored locally, with quick access links.
- **Landing Page Upgrade**: New marketing copy highlighting "Why Astra", "Features", and "Enterprise Security".

### Changed
- **Local Storage Architecture**: Re-architected storage to support an array of decisions rather than a single active instance, enabling Dashboard and Timeline features.
