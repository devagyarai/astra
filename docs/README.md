# Astra Documentation Architecture

Welcome to the Astra Documentation repository. This folder structure organizes all engineering, product, and architectural documentation for the Astra Decision Intelligence Platform.

## Directory Structure

- **[Reports](./reports/)**: Contains generated engineering logs, sprint summaries, and quality gate reports.
- **[Architecture](./architecture/)**: Technical specifications, system overviews, and infrastructure diagrams.
- **[Decisions](./decisions/)**: Architecture Decision Records (ADRs) that document significant technical choices and their context.
- **[Product](./product/)**: Product requirements, user flows, personas, and feature roadmaps.
- **[Research](./research/)**: Exploratory work, competitor analysis, UX findings, and AI experimentation.
- **[Assets](./assets/)**: Static media, diagrams, and design files supporting the documentation.
